Product Promo Ribbons and badges

This module will install elegant badges and ribbons to be used to promote your website products by tagging them with promotional messages such as discount offers, limited stocks, new products and many more promotions you want to run

Feature: Choose from several kinds of badges and ribbons

Currently there are three types of shapes/objects to choose from:

  1. 12 point burst shape
  2. Badge
  3. TV shape


Feature: Easy to change position, size and colors of shapes/objects and promotion text!

The sales team is able to change the size of each shape, the color of the shape and even postion of the shape and promotion text with respect to the product image. You can place the badge or ribbon or text on practically anywhere on the website. Unlimited number of colors to choose from by simply picking colors by its HTML code

Settings: How to add product style, change size, colors, text and shapes

Product styles are configurble from the sales menu as a sales manager. You need to go to Sales -> Configuration -> Sales -> Product Style. From there you can do the folwoing:

  1. Create a product style
  2. Pick the right object/shape to use
  3. Change position settings
  4. Change promo text
  5. Change promo text color settings
  6. Change promo text font settings
  7. Change object color settings
  8. Change object size settings
You can test as you change and once you are satisfied, you can then apply the style on any website product
